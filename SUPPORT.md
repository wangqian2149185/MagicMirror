# MagicMirror Support

For help with MagicMirror, contact the developer at wangqian2149185@yahoo.com.

## Common Notes

- MagicMirror can run in Free Local mode without an API key.
- API keys are optional and are stored locally on the device.
- Voice input is optional and requires microphone and speech-recognition permission.
- Generated personality and MBTI-style results are for reference only and are not diagnostic or professional advice.

## Troubleshooting

If cloud model analysis fails, check that the selected provider, model, API key, quota, and billing status are correct. You can also switch to Free Local mode to complete the interview without a model provider.
